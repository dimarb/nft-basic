# nft-basic

## Stack

- nodejs
- Hardhat 
- OpenZeppelin
- Solidity

## Guide

- Execute npm install
- Create .env from .emvExample
  - Custom environment params API_URL and PRIVATE_KEY
- Execute npx hardhat compile
- Execute  npx hardhat --network rinkeby  run scripts/deploy.js
